# how bad is it?

A tool that analyzes your Claude API usage stats and tells you just how deep you've gone.

Drop in your Claude Code usage JSON and get a severity diagnosis, daily activity breakdown, cost estimates across models, and a heatmap of your usage over time.

Don't use Claude Code? Take the self-survey instead: answer a few questions about your usage habits and get diagnosed anyway.

## Run locally

Open `index.html` in a browser, or serve it:

```sh
python -m http.server 8000
```
